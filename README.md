   cin2) 현금 입금 : 현금을 받은 후 진짜 지폐인지 확인하고, 계좌에 돈이 증가됨
		    시간 부족으로 구현 못함 진짜 지폐인지 확인은 하드웨어적으로......
   cin3) 현금 인출 : 계좌에 돈이 빠져나오고 계좌에 돈이 감소됨 / 잔액보다 인출할 금액이 크다면 인출 불가능함
       ) 디폴트도 넣어줌
